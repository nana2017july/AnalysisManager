typedef struct CAnalysisExecutor_HtmlTagReplace CAnalysisExecutor_HtmlTagReplace

CAnalysisExecutor の派生クラス。指定のHTMLタグを置換する。
このクラスの動作内容。
タグの開始（＜）を見つけたら、AC_POSでタグの終了（＞）を見つける。
その後、タグ自体を置換する場合はAC_REPLACEを実行。
タグの後ろを置換する場合はAC_FORWARDでタグの終了位置まで移動してから、AC_REPLACEを実行。
