Detailed Description

CAnalysisExecutor の派生クラス。マルチパートのリクエストを解析してパラメタを取得する。
現状ではファイル（filenameがヘッダに存在するもの）は取得しない。メモリを多く消費してしまうためである。
multipart/mixedのような二重構造には対応していない。
取得したパラメタの名前と値の文字列の長さには最大桁数があり、最大桁数 MULTIPART_NAME_MAX_LEN , MULTIPART_VALUE_LEN と同じ長さの場合、
もっと長い可能性もある。
